Design System · 2026

디자인도 감이 아니라, 규칙으로.Design by rules,
not by gut.

이 포트폴리오를 만든 규칙을 한곳에 모았어요. 색, 글자, 간격, 그리드, 컴포넌트, 움직임까지요. 값은 전부 한 군데(토큰)에 저장돼 있어서, 거기만 바꾸면 사이트 전체가 한 번에 바뀝니다. 그래서 화면마다 디자인이 들쭉날쭉하지 않아요.These are the rules behind this portfolio, all in one place — color, type, spacing, grid, components, motion. Every value is stored in one spot (a token), so changing it there updates the whole site at once. That's why the design never looks different from one screen to the next.

모든 값을 한곳에서 관리해요 · 칩을 누르면 이름이 복사됩니다Every value managed in one place · tap a chip to copy its name

00

원칙Principles principles

이 네 가지만 정해두면, 나머지 규칙은 저절로 따라와요.Decide these four, and every other rule falls into place.

1 · 꼭 필요한 것만 남기기1 · Keep only what's needed

꾸미는 대신 여백과 크기 차이로 보여줘요. 색은 거의 흑백, 강조는 네이비와 골드 딱 둘. 화면이 좀 비어 보여도 일부러 그렇게 둔 거예요.Instead of decoration, it leans on whitespace and size contrast. Color stays near black-and-white; emphasis is just two — navy and gold. If a screen looks a little empty, that's on purpose.

2 · 한국어를 먼저 생각한 타이포2 · Typography that thinks in Korean first

한글은 단어 중간에서 줄이 끊기면 읽기 불편해요. 그래서 띄어쓰기 단위로만 넘어가게(keep-all) 해뒀어요. 글꼴도 한글은 Pretendard, 영문 제목은 Fraunces로 나눠, 두 언어가 각자 가장 예쁜 모습이 되게 했어요.Korean gets hard to read when a word splits at the end of a line, so lines break only at spaces (keep-all). Fonts split too — Korean in Pretendard, English headings in Fraunces — so each language looks its best.

3 · 4px 그리드 위의 간격3 · Spacing on a 4px grid

여백은 전부 4px의 배수예요. 눈대중으로 띄우지 않고 정해진 단계(--sp-1~--sp-10)에서만 고릅니다. 덕분에 어느 화면을 봐도 간격의 리듬이 똑같아요.Every margin is a multiple of 4px. Nothing is spaced by eye — values come only from a set scale (--sp-1–--sp-10). So the spacing rhythm feels the same on every screen.

4 · 이중 언어, 하나의 구조4 · Two languages, one structure

버튼 하나로 한국어와 영어를 오가도, 레이아웃·간격·컴포넌트는 그대로예요. 언어만 바뀔 뿐, 같은 공간에 머무는 느낌을 줍니다.Flip between Korean and English with one button, and the layout, spacing, and components stay put. Only the language changes — you never leave the same room.

01

색Color color

바탕은 따뜻한 오프화이트, 글자는 검정에 가까운 색이에요. 눈길을 끄는 색은 네이비(링크·중요한 숫자)와 골드(아직 못 채운 자리), 딱 둘뿐입니다. 회색은 세 단계로 나눠 무엇이 더 중요한지 보여줘요.A warm off-white ground, near-black text. Only two colors grab attention: navy (links, key numbers) and gold (spots still to fill). Grey comes in three steps to show what matters more.

간격Spacing spacing

여백은 전부 4px의 배수예요. 아래 단계에서만 값을 고르니까, 버튼 안쪽 여백부터 섹션 사이 간격까지 전부 같은 리듬을 탑니다.Every margin is a multiple of 4px. Values come only from the steps below, so everything — from the padding inside a button to the gap between sections — moves to the same rhythm.

모션Motion motion

움직임은 거들 뿐, 주인공은 아니에요. 모든 움직임이 같은 속도 곡선(--ease) 하나를 써서, 어디서나 같은 결로 느껴집니다.Motion plays a supporting role, never the lead. Everything moves on one shared easing curve (--ease), so it all feels of a piece.

패턴Patterns

등장 — 화면에 들어오면 22px 아래에서 슥 떠오르며 0.8초간 나타나요. 여럿이 묶이면 0.08초씩 차례로(d1~d4).
숫자 세기 — 성과 숫자가 0에서 목표값까지 1.1초에 걸쳐 차오릅니다.
스냅 스크롤 — 홈에서는 한 화면씩 부드럽게 딱 멈춰요(scroll-snap-type: y proximity).
마우스 올리기 — 카드는 4px 떠오르고, 화살표와 링크는 0.2~0.35초에 걸쳐 바뀝니다. Reveal — when it enters the screen, it rises 22px and fades in over 0.8s. Grouped items follow one after another, 0.08s apart (d1–d4).
Count-up — result numbers climb from 0 to their target over 1.1s.
Snap scroll — on the home, it settles one screen at a time (scroll-snap-type: y proximity).
Hover — cards lift 4px; arrows and links shift over 0.2–0.35s.

모두를 위해: 기기에서 '움직임 줄이기(prefers-reduced-motion)'를 켠 분께는 애니메이션과 스냅을 꺼둡니다. 움직임이 불편한 분을 위한 기본 배려예요.For everyone: if a device has 'reduce motion' (prefers-reduced-motion) turned on, animation and snap switch off — a small courtesy for anyone who's motion sensitive.
